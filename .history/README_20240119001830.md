This is google clone build with react js. The core functionalities of this build include:

-

## Tech Stack

- React JS
- React Context API
- React Router
- React Hooks
- Material UI core/icons
- Tailwind CSS
- Firebase google authentication
- Firebase cloud firestore

`$ cd google-clone` <br/>
